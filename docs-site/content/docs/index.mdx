## Quick Start

Install the package:

```bash npm2yarn
npm install dhan-ts
```

Initialize the client:

```typescript
import { DhanHqClient, DhanFeed, DhanEnv } from 'dhan-ts';

const config = {
  accessToken: 'your-access-token',
  clientId: 'your-client-id',
  env: DhanEnv.PROD,
};

const dhanClient = new DhanHqClient(config);
const dhanFeed = new DhanFeed(config);
```

## Main Components

### DhanHqClient - REST API

Access all REST API endpoints through organized modules:

```typescript
// Orders
await dhanClient.orders.getOrders();
await dhanClient.orders.placeOrder(orderRequest);

// Portfolio
await dhanClient.portfolio.getHoldings();
await dhanClient.portfolio.getPositions();

// Market Data
await dhanClient.marketData.getLTP(request);
await dhanClient.marketData.getProcessedCandleData(request);

// Funds
await dhanClient.funds.getFundLimit();
```

### DhanFeed - WebSocket Feeds

Real-time market data and order updates:

```typescript
// Live market feed
const feed = dhanFeed.liveFeed;
await feed.connect();
feed.subscribe(instruments, FeedRequestCode.SUBSCRIBE_TICKER);

feed.on('data', (data) => {
  console.log('Market data:', data);
});

// Live order updates
const orderFeed = dhanFeed.liveOrderUpdate;
await orderFeed.connect();
```

## Available Modules

| Module | Description |
|--------|-------------|
| `orders` | Place, modify, cancel, and query orders |
| `portfolio` | Get holdings, positions, and convert positions |
| `marketData` | Real-time and historical market data |
| `funds` | Fund limits and margin calculator |
| `authentication` | App-based and partner authentication |
| `superOrders` | Advanced order types with targets and stop losses |
| `foreverOrders` | GTT (Good Till Triggered) orders |
| `optionChain` | Option chain data and Greeks |
| `edis` | E-DIS authentication and inquiry |
| `statements` | Ledger and trade history |
| `tradersControl` | Kill switch functionality |
| `scanner` | Market scanner for equity and F&O |

## WebSocket Feeds

| Feed | Description |
|------|-------------|
| `liveFeed` | Real-time market data (ticker, quote, full) |
| `liveOrderUpdate` | Real-time order status updates |
| `marketDepthFeed20` | Level 2 market depth (20 levels) |
| `marketDepthFeed200` | Level 2 market depth (200 levels) |
| `multiConnectionLiveFeed` | Multiple concurrent connections |
| `mockLiveFeed` | Testing without live connection |
